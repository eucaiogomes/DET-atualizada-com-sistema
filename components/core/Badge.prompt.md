Compact mono-uppercase status chip for Juris8 — statuses, urgency tags, trust seals.

```jsx
<Badge tone="ok" dot>Monitorando</Badge>
<Badge tone="bad">Urgente</Badge>
<Badge tone="neutral">LGPD</Badge>
```

Tones: `brand` (default), `neutral`, `ok`, `warn`, `bad`, `info`. Set `dot` for a leading status dot (inherits the tone color). Always mono, always uppercase — never use for long text.
