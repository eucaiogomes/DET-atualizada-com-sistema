Labelled text field for Juris8 forms — raised surface, violet focus ring.

```jsx
<Input id="f-email" label="E-mail profissional" type="email" placeholder="voce@escritorio.com.br" />
<Input id="f-msg" label="Mensagem" textarea placeholder="Conte sobre sua carteira…" />
```

Props: `label`, `hint`, `textarea` (multi-line), plus all native input attributes. Pair with `Select` for dropdowns.
